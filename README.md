# Prodigy_AD_04
# Tic Tac Toe Android App

A simple Tic Tac Toe game for Android, allowing two players to compete on a 3x3 grid.

## Features
- Two-player mode
- Automatic win and draw detection
- Reset button to restart the game

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/Sakshi-Ambekar/Prodigy_AD_04.git

2. Open with Android Studio:
- Open Android Studio.
- Select Open an existing project.
- Navigate to the cloned directory and open it.

3. Build and run the app on an emulator or device.

## Usage
- Players take turns tapping the grid cells to place their marks (X or O).
- The game checks for a winner or a draw after each move.
- Press the reset button to start a new game.
